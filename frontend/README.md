## Requisitos
- Node.js 20+

## Desarrollo local
```bash
npm install
npm run dev
```

## Variables de entorno
Copia `frontend/.env.example` a `frontend/.env.local` y completa valores reales.

## Build
```bash
npm run build
npm run start
```

## Deploy en Vercel
- Root Directory: `frontend`
- Framework: Next.js
- Build command: `npm run build`
- Install command: `npm install`
- Variables: usar las de `frontend/.env.example`
